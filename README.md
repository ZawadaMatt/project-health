# Project Health


Project Health was created in cooperation of 4 friends as a final project of "Java from the basics" carried out by Software Development Academy, for which we participated in the period from May to December 2019.
Our application is platform for the Dietician for managing his patients,  allowing him to create a diet based on defined ingredients.
### Technology
- Maven
- Spring Boot
- Liquibase
- Thymeleaf
- Spring Security
- MySQL Driver 
- Spring Data JPA
- Spring Web
- JUnit
